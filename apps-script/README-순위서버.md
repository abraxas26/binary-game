# 🐍 스네이크 게임 순위 서버 설치 방법

스네이크 게임의 순위(랭킹)를 구글 스프레드시트에 저장하는 서버입니다.
아래 순서대로 하면 5분 안에 연동됩니다.

## 1. 스프레드시트 만들기
1. [Google Sheets](https://sheets.google.com) 에서 **빈 스프레드시트**를 새로 만듭니다.
   (이름은 자유, 예: `스네이크 순위`)

## 2. Apps Script 코드 붙여넣기
1. 그 스프레드시트에서 상단 메뉴 **확장 프로그램 → Apps Script** 클릭
2. 기본으로 열린 `Code.gs` 내용을 모두 지우고,
   이 폴더의 [`snake-ranking.gs`](snake-ranking.gs) 내용을 **전부 복사해서 붙여넣기**
3. 💾 저장 (Ctrl+S)

## 3. 웹앱으로 배포하기
1. 우측 상단 **배포 → 새 배포** 클릭
2. 톱니바퀴(⚙️) → **웹 앱** 선택
3. 설정:
   - **실행 계정**: `나(본인)`
   - **액세스 권한**: **`모든 사용자`** ← ⚠️ 꼭 이걸로!
4. **배포** 클릭 → 권한 승인(본인 구글 계정으로 허용)
5. 나오는 **웹 앱 URL** 을 복사합니다.
   (`https://script.google.com/macros/s/.../exec` 형태)

## 4. 게임에 주소 넣기
1. `index.html` 파일을 엽니다.
2. 아래 줄을 찾습니다 (`SNAKE_ENDPOINT`):
   ```js
   const SNAKE_ENDPOINT = "";
   ```
3. 따옴표 안에 복사한 웹 앱 URL을 붙여넣습니다:
   ```js
   const SNAKE_ENDPOINT = "https://script.google.com/macros/s/여기에붙여넣기/exec";
   ```
4. 저장하고 GitHub에 push 하면 끝! 🎉

## 확인
- 게임을 한 판 하고 **이름을 넣어 "순위 등록"** 하면 스프레드시트 `ranking` 시트에 줄이 추가됩니다.
- 게임 오버 화면의 **🏆 TOP 10** 에 실시간 순위가 표시됩니다.

## 참고
- 주소를 넣기 **전**에는 순위가 그 기기(브라우저)에만 저장됩니다(localStorage). 주소를 넣으면 모든 사람이 공유하는 순위가 됩니다.
- 코드를 수정한 뒤에는 **배포 → 배포 관리 → 편집(연필) → 새 버전 → 배포** 로 다시 배포해야 반영됩니다.
